# Robotyka Ewolucyjna

## Instalacja

1. Sklonuj repozytorium:
   ```bash
   git clone https://github.com/twoj-user/robotyka-ewolucyjna.git
   cd robotyka-ewolucyjna
   ```

2. Utwórz i aktywuj wirtualne środowisko:

   **Linux / macOS**
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```

   **Windows**
   ```bash
   python -m venv venv
   venv\Scripts\activate
   ```

3. Zainstaluj zależności:
   ```bash
   pip install -r requirements.txt
   ```

## Uruchomienie

```bash
python src/main.py
```
